## 🚀 Project Overview

**CloudNova** is a modern static website developed using **HTML5, CSS3, and JavaScript** as part of my **Cloud Computing Internship**.

The project demonstrates how to build, manage, and deploy a static website using **Amazon S3 Static Website Hosting**. To ensure continuous public access after my AWS Free Plan ended, the project is also deployed on **Netlify**.
